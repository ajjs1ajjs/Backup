## 🎓 ІНСТРУКЦІЇ

### Швидкий старт
1. Розпакуйте `NovaBackup-4.0.0-Setup.zip`
2. Запустіть `SimpleInstaller.bat` від адміна
3. Знайдіть ярлик на робочому столі
4. Двічі клікніть для запуску

### Створення Backup Job
1. Home → Backup Job
2. Введіть ім'я (напр. "Daily Backup")
3. Додайте файли/папки
4. Оберіть сховище
5. Налаштуйте розклад
6. Збережіть

### Запуск Backup
1. Виділіть job в списку
2. Jobs → Run
3. Спостерігайте за прогресом

### Перегляд результатів
1. Monitoring → Sessions
2. Dashboard → Statistics
3. Repositories → Usage
