# 📦 NovaBackup - Інструкція з встановлення

## ✅ MSI Інсталятор (Повноцінний)

### Файл: `installer\NovaBackup-1.0.0.msi` (22 MB)

**Особливості:**
- ✅ Перевіряє чи вже встановлено (не дозволить встановити двічі)
- ✅ Показує прогрес встановлення
- ✅ Встановлює в `C:\Program Files\NovaBackup`
- ✅ Створює ярлики в меню Пуск і на робочому столі
- ✅ Додає запис у "Програми та компонії" для видалення
- ✅ Вимагає .NET Desktop Runtime 8.0 (якщо немає - запропонує встановити)

### Встановлення:

**Спосіб 1 - Двічі клацнути:**
1. Відкрийте папку `installer`
2. Двічі клацніть на `NovaBackup-1.0.0.msi`
3. Натисніть "Next" → "Install"
4. Готово!

**Спосіб 2 - Командний рядок:**
```cmd
# Тихе встановлення з прогресом
msiexec /i installer\NovaBackup-1.0.0.msi

# Повністю тихе встановлення (без вікон)
msiexec /i installer\NovaBackup-1.0.0.msi /quiet

# Встановлення з логуванням
msiexec /i installer\NovaBackup-1.0.0.msi /l*v install.log
```

### Видалення:

**Через Панель керування:**
1. Панель керування → Програми та компоненти
2. Знайдіть "NovaBackup"
3. Натисніть "Видалити"

**Командний рядок:**
```cmd
msiexec /x {ProductCode} /quiet
```

## 📦 ZIP Архів (Portable версія)

### Файл: `installer\NovaBackup-1.0.0.zip` (84 MB)

**Особливості:**
- ✅ Не потребує встановлення
- ✅ Містить все необхідне (включно з .NET Runtime)
- ✅ Працює з будь-якої папки
- ✅ Можна видалити просто видаливши папку

### Встановлення:
1. Розпакуйте ZIP в `C:\Program Files\NovaBackup` або будь-яку іншу папку
2. Запустіть `NovaBackup.GUI.exe`
3. Готово!

## 🔧 Вимоги

### Для MSI (22 MB):
- Windows 10/11 x64
- **.NET Desktop Runtime 8.0** (якщо немає -系统将 запропонувати встановити)
- 200 MB вільного місця

### Для ZIP (84 MB):
- Windows 10/11 x64
- 300 MB вільного місця
- **НЕ вимагає** .NET Runtime (все включено)

## ❓ Усунення проблем

### "A newer version is already installed"
Інсталятор виявив вже встановлену версію. Спочатку видаліть стару версію.

### "You must install .NET Desktop Runtime"
Завантажте та встановіть: https://dotnet.microsoft.com/download/dotnet/8.0

Або використайте ZIP версію (84 MB) - вона не вимагає Runtime.

### Встановлення зависає
1. Відкрийте Диспетчер завдань
2. Знайдіть "Windows Installer"
3. Завершіть процес
4. Спробуйте знову

### Помилка при видаленні
Використайте утиліту Microsoft Install Troubleshooter:
https://support.microsoft.com/help/17588/windows-fix-problems-that-block-programs-being-installed-or-removed
