===========================================
NovaBackup v7.0 Enterprise - Read Me
===========================================

Thank you for installing NovaBackup v7.0!

===========================================
INSTALLATION
===========================================

1. Run the installer (NovaBackup-Enterprise-Setup.exe)
2. Follow the installation wizard
3. Launch NovaBackup from Start Menu or Desktop shortcut

===========================================
SYSTEM REQUIREMENTS
===========================================

- Windows 10/11 x64 or Windows Server 2019+
- 4 GB RAM minimum (8 GB recommended)
- 500 MB free disk space
- .NET 8.0 Runtime (included)

===========================================
WHAT'S NEW IN v7.0
===========================================

✨ Synthetic Full Backups
   - Create synthetic full backups from incrementals
   - Merge incremental backups efficiently
   - No impact on production systems

🔐 Encryption at Rest (AES-256-GCM)
   - Military-grade encryption
   - Hardware acceleration (AES-NI)
   - GDPR & HIPAA compliant

🖥️ Modern WPF Interface
   - Veeam-inspired design
   - Dark theme
   - Real-time monitoring

📊 Enterprise Features
   - RBAC (Role-Based Access Control)
   - Audit Logging
   - VSS Integration
   - Guest Processing
   - Replication
   - CDP (Continuous Data Protection)

===========================================
QUICK START
===========================================

1. Launch NovaBackup
2. Create a Backup Job:
   - Home → Backup Job
   - Add files/folders
   - Select repository
   - Configure schedule
   - Save and Run

3. Monitor Progress:
   - Dashboard shows real-time status
   - View sessions in Monitoring tab
