# NovaBackup v6.0 - GUI Installer Options 
  
## Option 1: Inno Setup (Recommended)  
  
1. Download Inno Setup: https://jrsoftware.org/isdl.php  
2. Open: novabackup_full.iss  
3. Click: Build  
4. Output: output\NovaBackup-6.0.0-Setup.exe  
  
## Option 2: WinForms GUI  
  
1. Open Visual Studio 2022  
2. Open: NovaBackupInstaller\NovaBackupInstaller.csproj  
3. Build  
4. Output: bin\Release\NovaBackupInstaller.exe  
  
## Option 3: Portable (No Install)  
  
Simply run: dist\menu.bat
