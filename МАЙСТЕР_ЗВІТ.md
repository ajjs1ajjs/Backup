## 🚀 ЯК ПОЧАТИ (3 КРОКИ)

### Крок 1: Оберіть версію

**Повна установка** (для постійного використання):
```
1. Розпакуйте NovaBackup-4.0.0-Setup.zip
2. Запустіть SimpleInstaller.bat ВІД ІМЕНІ АДМІНІСТРАТОРА
3. Готово!
```

**Портативна** (для тестування/USB):
```
1. Розпакуйте NovaBackup-4.0.0-Portable.zip
2. Запустіть RunNovaBackup_v4.bat
3. Готово!
```

### Крок 2: Запустіть програму

```
Спосіб 1: Двічі клікніть на ярлик NovaBackup
Спосіб 2: Пуск → NovaBackup
Спосіб 3: RunNovaBackup_v4.bat
```

### Крок 3: Створіть перший backup

```
1. Home → Backup Job
2. Введіть ім'я
3. Додайте файли
4. Оберіть сховище
5. Збережіть
6. Jobs → Run
```
