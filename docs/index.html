Core Features

Everything you need for enterprise backup management

🖥️

Virtual Machine Support

Inventory and backup hypervisors (Hyper-V) and virtual machines with metadata tracking.

📦

Repository Management

Configure and manage backup repositories with connection testing and storage tracking.

📅

Job Scheduling

Create and schedule backup jobs with manual triggering and status monitoring.

🔄

Restore Operations

Browse and restore backups with point-in-time recovery capabilities.

📊

Reports & Analytics

Dashboard with activity summaries, backup statistics, and exportable reports.

🔒

Security & Auth

JWT-based authentication with role-based access control and password policies.

Quick Installation

Get started in minutes

Windows (PowerShell as Administrator)

iwr -useb https://raw.githubusercontent.com/ajjs1ajjs/Backup/main/install-server.ps1 -OutFile install-server.ps1
.\install-server.ps1 -AutoStart

Access UI at http://localhost, API at http://localhost:8000
